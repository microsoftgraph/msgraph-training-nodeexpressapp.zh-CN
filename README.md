# <a name="microsoft-graph-training-module---build-nodejs-express-apps-with-microsoft-graph"></a>Microsoft Graph培训模块 - 使用 Microsoft Node.js生成 Express Graph

本模块将向您介绍使用 Microsoft Graph，以通过生成 Office 365 Express Node.js访问数据。

## <a name="lab---build-nodejs-express-apps-with-microsoft-graph"></a>实验室 - 使用 Microsoft Node.js生成 Express Graph

在此实验中，你将使用 Azure AD v2 身份验证终结点创建 Node.js Express Web 应用程序，以使用 Microsoft Office 365 访问 Graph。

- [Node.js Microsoft Graph教程](https://docs.microsoft.com/graph/training/node-tutorial)
